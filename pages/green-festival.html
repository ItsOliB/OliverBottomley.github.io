Bauhaus - College Website Redesign

College Work - This range of work was produced during my time on Level 3 Graphic Design.

This piece was made for the Green Festival Brief. I was tasked to create the branding, logo and products for the festival. This can include any products that would sell as well as a poster, banner, etc...

Design Process:

In this design, I changed the colour of the F and added All of the same effects and then merged it into the green text and added a few of the effects again on it so it would look the same type of way and like it was not modified and changed like that. I then added text next to the F in white with the same effects and then I moved it closer so you would never know it was changed like it is and it was different text. So I added bevel & emboss, drop shadow and a stroke to both parts of the text, this gives it a more 3D look and adds depth to both text. I then added a dark solid background which makes the text stand out. I think next time I would do a typography type logo, and try a different style of design
