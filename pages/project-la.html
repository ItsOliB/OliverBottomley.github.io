Project LA Branding

Graphic Design - This range of work is logo/branding design pieces.

This branding was created for a personal project I was working on called Project LA. The purpose of the branding was made for a FiveM server which I was operating at the time. The colours are based off the Dodges baseball team "LA logo" and common colours seemed to be used in Los Angeles. I went for a caligraphy style typeface as I wanted it to fit with the theme of the server, which is California based. I then added a white stroke style behind it to make sure both text was outlined and flowed together as the logo.
